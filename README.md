# Projet "Trier avec SHELL"

Projet réalisé par Charles Kempa, Thibaut Masnin et Thomas Dignoire.

## Description

Programme qui trie les entrées d’un répertoire suivant différents critères. Ce programme est réalisé en SHELL.
